# Student Companion App

A comprehensive platform designed for students to collaborate and manage their academic life efficiently. 
Built with React, Tailwind CSS, and Appwrite, the app provides a seamless and modern user experience.

- **Real-time Communication**: Chat with classmates, share files, and send playable voice notes.
- **Voice Calls**: Secure 1-on-1 peer-to-peer voice calls powered by WebRTC.
- **Contacts Directory**: Search and connect with classmates by name, email, or matric number.
- **Help Desk**: Ask academic questions and receive real-time answers from the community.
- **Academic Tools**: Manage your assignments and keep track of your class timetable.
- **Cross-Platform**: Designed to work perfectly on both mobile devices and desktops.
